# Эксплуатационные требования

Этот репозиторий не является промышленным дистрибутивом DataHub.

Для промышленного или корпоративного внедрения владелец DataHub должен отдельно
определить и обеспечить:

- TLS termination и доверенные сертификаты;
- аутентификацию и SSO;
- модель авторизации;
- резервное копирование и восстановление хранилищ DataHub;
- мониторинг и оповещения;
- подбор ресурсов для Kafka, OpenSearch и SQL-хранилища;
- безопасное хранение секретов;
- отказоустойчивость;
- политику обновления DataHub и расширения;
- pipeline развёртывания сервиса метаданных 1С;
- обработку лицензий для опциональной автоматизации 1C Designer.

Демонстрационный контур - это воспроизводимое окружение для демонстраций и
пилотных проверок. Он нужен для проверки сценария работы расширения, а не для
замены эксплуатационных решений.
