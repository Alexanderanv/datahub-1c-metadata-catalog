# Расширение DataHub для 1С

Эта документация описывает установку и запуск расширения, которое добавляет в
DataHub поддержку метаданных 1С:Предприятия.

2. [Установка расширения](install-datahub-extension.md) - установка коннектора
   и кастомной метамодели.
3. [Быстрый демонстрационный запуск](quickstart-demo.md) - локальный DataHub
   quickstart для первого просмотра результата.

   коннектор.
5. [ConfigDumpInfo](configdumpinfo.md) - источник стабильных UUID метаданных 1С.
6. [Recipes](recipes.md) - структура YAML-файлов загрузки.
7. [Запуск загрузки](ingestion.md) - как выполнять загрузочный процесс.
8. [Демонстрационный контур](reference-deployment.md) - минимальный контур для
   демонстраций и пилотных проверок.

## Область применения

Расширение добавляет поддержку метаданных 1С в уже существующую установку
DataHub. Оно не заменяет проектирование и эксплуатацию самого DataHub.

Локальный DataHub quickstart и демонстрационный контур в `deploy/reference`
предназначены для демонстраций и пилотной проверки. Они намеренно минимальны и
не закрывают весь набор эксплуатационных требований промышленного окружения.
